TECNOLOGIAS

HTML5

HTML5 é uma linguagem de marcação para a construção de páginas web. Inclui recursos novos como suporte a vídeo e áudio nativo, melhorias na estruturação de conteúdo, entre outros.

CSS3

CSS3: Linguagem para estilizar páginas web. Adiciona novos recursos, como animações, efeitos e adaptações para dispositivos, para melhorar o layout e visualização.

JS

JS: Linguagem interpretada para criar interações e efeitos dinâmicos nas páginas web. Possibilita manipular elementos HTML e CSS e acessar recursos do navegador.

REACT-JS

React: Biblioteca JavaScript para construir interfaces de usuário. Ele se concentra em componentes reutilizáveis, tornando o desenvolvimento mais fácil e escalável baseada em componentes.

NODE-JS

Node.js é um ambiente de execução JavaScript baseado no motor V8 do Google Chrome. Node: Ambiente para executar JavaScript no lado do servidor e usar recursos do sistema operacional.

MONGO-DB

MongoDB: Banco de dados NoSQL baseado em documentos, flexível e escalável. o que significa que armazena dados em documentos em vez de tabelas relacionais.

DOCKER

Docker: Plataforma de containerização para embalar, enviar e executar aplicativos em qualquer lugar. Utiliza containers para garantir consistência e confiabilidade em diferentes ambientes.
